1. Question: Reverse Words in a String
Write a function reverseWords that takes a string as input and reverses the order of words in it while maintaining the order of characters within each word.

Input: "Hello World"
Output: "World Hello"

2. Question: Count Character Frequencies
Implement a function countCharacters that takes a string as input and returns a dictionary where keys are characters and values are their frequencies in the string. Ignore spaces.

Input: "hello world"
Output: {'h': 1, 'e': 1, 'l': 3, 'o': 2, 'w': 1, 'r': 1, 'd': 1}

3. Question: Check Anagrams
Write a function areAnagrams that takes two strings as input and returns true if they are anagrams of each other, false otherwise. Ignore spaces and consider case insensitivity.

Input: "listen", "silent"
Output: true

4. Question: Longest Palindromic Substring
Implement a function longestPalindrome that finds the longest palindromic substring in a given string.

Input: "babad"
Output: "bab" (or "aba")

5. Question: Reverse String
Write a function reverseString that takes a string as input and returns the string reversed.

Input: "hello"
Output: "olleh"
